Spring Boot + Flask AI

건강검진 데이터 기반 비만 예측 서비스

키와 몸무게를 입력하면 Flask AI 서버가 머신러닝 모델로 비만 여부를 예측하고, Spring Boot 화면에서 결과를 확인할 수 있습니다.

AI 예측 시작 게시판 보기

서비스 흐름

  1. 사용자가 키와 몸무게를 입력합니다.
  2. Spring Boot 서버가 Flask API를 호출합니다.
  3. 예측 결과를 화면에 출력합니다.
주요 기능

포트폴리오 핵심 기능

회원/로그인

Spring Security 기반 formLogin과 BCrypt 비밀번호 암호화를 적용했습니다.

게시판
