# 요구사항 정의서

## 프로젝트 목적

건강검진 데이터를 기반으로 사용자의 비만 여부를 예측하는 AI 웹 서비스를 개발합니다.

Spring Boot와 Flask 서버를 분리하여 REST API 기반 서버 연동 구조를 학습하고, 머신러닝 모델을 웹 서비스에 연결하는 경험을 목표로 합니다.

## 사용자 요구사항

### 비회원 사용자

- 키와 몸무게 데이터를 입력할 수 있어야 합니다.
- AI 예측 결과를 확인할 수 있어야 합니다.

### 회원 사용자

- 회원가입 및 로그인이 가능해야 합니다.
- 게시판 기능을 사용할 수 있어야 합니다.

## 시스템 요구사항

### Spring Boot 서버

- 사용자 요청을 처리합니다.
- Thymeleaf 기반 화면을 렌더링합니다.
- Flask AI 서버 API를 호출합니다.
- 게시판 CRUD를 처리합니다.
- Spring Security 기반 인증을 처리합니다.

### Flask AI 서버

- 머신러닝 모델을 로드합니다.
- 입력 데이터를 기반으로 예측을 수행합니다.
- 예측 결과를 JSON 형식으로 반환합니다.

## 예측 기능 요구사항

### 입력값

- 키
- 몸무게

### 출력값

- 비만 여부 예측 결과

## 비기능 요구사항

- 반응형 UI를 지원합니다.
- REST API 기반 통신을 사용합니다.
- Bootstrap UI를 적용합니다.
- GitHub를 이용해 형상관리를 수행합니다.
