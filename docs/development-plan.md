# 개발 계획서

## 1단계 - 프로젝트 초기 설정

- Spring Boot 프로젝트를 생성합니다.
- Flask 프로젝트를 생성합니다.
- GitHub 저장소를 생성합니다.
- README 및 프로젝트 문서를 작성합니다.

## 2단계 - 백엔드 기본 기능 구현

- 회원가입 기능을 구현합니다.
- 로그인 기능을 구현합니다.
- Spring Security를 적용합니다.
- 게시판 CRUD 기능을 구현합니다.

## 3단계 - AI 서버 구현

- 건강검진 데이터를 전처리합니다.
- 머신러닝 모델을 학습합니다.
- Flask 예측 API를 구현합니다.

## 4단계 - 서버 연동

- Spring Boot 서버에서 Flask API로 요청을 보냅니다.
- JSON 데이터를 처리합니다.
- 예측 결과를 화면에 출력합니다.

## 5단계 - 프론트 UI 개선

- Bootstrap 기반 반응형 UI를 적용합니다.
- 메인 페이지를 구성합니다.
- 예측 화면 UI를 개선합니다.

## 6단계 - 테스트 및 배포

- 주요 기능을 테스트합니다.
- 예외 처리를 보완합니다.
